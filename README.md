# 🚗 Car Price Prediction

This project uses **Linear Regression** to predict the selling price of used cars based on different features such as the car's year, present price, kilometers driven, and owner details.

## 📂 Files

* `car_price.ipynb` – Jupyter Notebook containing the complete project
* `car data.csv` – Dataset used for training and testing the model

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Workflow

* Load the dataset
* Clean and preprocess the data
* Perform Exploratory Data Analysis (EDA)
* Split the data into training and testing sets
* Train a Linear Regression model
* Evaluate model performance
* Predict car prices

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

   ```
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open `car_price.ipynb` in Jupyter Notebook.
4. Run all the cells.
